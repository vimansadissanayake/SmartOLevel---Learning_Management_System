SmartOLevel Learning Management System

Online Learning Management System (LMS) built for O-Level students, teachers, and administrators.

Overview

SmartOLevel LMS is a PHP-based web application that helps manage online learning for O-Level education. 

It supports:

- Students: View notes, watch videos, submit assignments, take quizzes, and check grades.
- Teachers: Upload notes & videos, create assignments & quizzes, grade work, and view results.
- Admins: Manage users, subjects, teacher assignments, student enrollments, and monitor content.

Technologies Used

- PHP (native)
- MySQL (database)
- Tailwind CSS (styling)
- phpMyAdmin (database management)

Requirements

- PHP 
- MySQL / MariaDB
- Web server (Apache)
- phpMyAdmin (for easy database setup)
